## Description

Chat written in [Nest](https://github.com/nestjs/nest) for whatsapp connection using [whatsapp-web]().js
and integration with rasa

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
